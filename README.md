# Machine Service Management System

Java desktop application for managing clients, machines, repairs, and payments.

## 🚀 Features
- Client management
- Employee management
- Machine tracking
- Repair tracking
- Payment system

## 🛠 Technologies
- Java (Swing)
- SQLite
- NetBeans

## ▶️ How to Run
1. Open the project in NetBeans
2. Run Login.java (main class)
